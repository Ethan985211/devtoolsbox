标题: [分享创造] DevToolsBox — 12款免费在线开发者工具，纯浏览器端处理，数据零上传

正文:
一直想找一个轻量、隐私友好、不用注册的在线工具箱，最后决定自己动手做了一个。

DevToolsBox 是一个纯静态的在线工具站，所有处理都在浏览器本地完成，数据不会上传到任何服务器。基于 Cloudflare Pages 部署，零运维成本，而且跑得飞快。

目前内置12款工具：
图片 · 压缩 / 格式转换(PNG↔JPG↔WebP↔BMP)
PDF · 合并 / 拆分 / 文本提取
编码 · JSON 格式化 / Base64 编解码 / URL 编解码
开发 · 二维码生成 / 时间戳转换 / 颜色转换 / 随机密码生成

技术特点：
- 纯前端，无后端，无数据库
- 暗色/浅色双主题，跟系统自动切换
- PWA 支持，可添加到桌面离线使用
- 二维码引擎完全自研（按ISO 18004标准实现纠错、掩码、格式信息编码）
- 所有图片/PDF处理均在浏览器本地完成

之前在网上淘了十几个工具站，要么注册才能用，要么疯狂弹广告，要么处理速度感人。所以干脆自己撸了一个，想要的工具自己加。

欢迎大家试用和反馈，有什么想加的工具也可以留言。
